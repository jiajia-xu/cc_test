# 2025AMS_6

This repository contains MATLAB simulations and generated results for ultrashort pulse analysis, spectrum calculation, Stokes parameters, and Poincare sphere trajectories.

## Folder Structure

- `code/`: MATLAB scripts.
- `figures/`: Generated figures.
- `data/`: CSV and XLSX output data.
- `media/`: Generated videos.
- `old_or_unused/`: MATLAB autosaves and figures that may not be used.

## Main Code Files

- `code/pulse250312.m`: pulse waveform and intensity calculations.
- `code/spectrum250312.m`: spectrum calculations for different pulse durations.
- `code/calculation_k_prime.m`: k-prime related calculation.
- `code/stokes/`: Ex/Ey, Stokes parameters, and Poincare sphere scripts.
- `code/bk7/`: BK7-related Stokes and Poincare sphere scripts.

## Figures

- `figures/pulse/`: pulse waveform and intensity figures.
- `figures/spectrum/`: spectrum figures.
- `figures/stokes/`: Stokes parameter figures.
- `figures/poincare_sphere/`: Poincare sphere trajectory figures.
- `figures/field_components/`: Ex/Ey field component figures.
- `figures/by_original_folder/`: all PNG figures copied with their original folder structure.

## Notes

This is a cleaned GitHub version of the original MATLAB working folder. The MATLAB source files were copied without changing their code.
